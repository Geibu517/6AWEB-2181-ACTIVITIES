# Dice App Repository

This repository contains different iterations of a simple **Dice App** built with Flutter.

---

## Project Versions

### 1. `dice_app`
- **Description:** The first iteration of the app.  
- **Features:** Includes only **one die** that can be rolled.

### 2. `dice_app_v2`
- **Description:** Second iteration of the app.  
- **Features:**  
  - Added a **second die**.  
  - Both dice roll **at the same time**.  
  - Shows the **total of both dice** after rolling.

### 3. `dice_app_v3`
- **Description:** Final version of the app.  
- **Features:**  
  - All previous features from `dice_app_v2`.  
  - Added modifications and improvements, such as:  
    - Dynamic total color based on value:
      - **Yellow** if total is 6  
      - **Red** if total is less than 6  
      - **Blue** if total is greater than 6  
    - Other UI/UX enhancements.
